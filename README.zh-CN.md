# server-node-bundle

服务器管理

## 安装

```bash
composer require tourze/server-node-bundle
```

## 使用方法

### 菜单集成

本包提供了 AdminMenu 服务，可以轻松集成到使用 `tourze/easy-admin-menu-bundle` 的应用中。无需额外配置，服务会自动注册。

菜单项包括：

- 服务器管理
  - 服务器节点
  - 应用管理
- 统计监控
  - 节点统计
  - 流量统计
